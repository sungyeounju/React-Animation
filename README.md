# framer-motion
Framer가 제공하는 리액트용 애니메이션 라이브러리

https://www.framer.com/motion/

## start
1. 설치 : npm install framer-motion
2. 가져오기 : import { motion } from "framer-motion"
3. 작성법 : 
```
<motion.div
  initial={{ scale: 0 }} 
  animate={{ scale: 1, rotateZ: 360 }}
/>
```
